## 3. 平台义务(ibi.ren)

### 3.1 算法备案

依据《互联网信息服务深度合成管理规定》第 19 条,提供深度合成服务的,应 **履行算法备案手续**:

- 🚧 平台当前未对外提供在线生成服务,仅做资产交易与撮合,**暂未触发强制备案义务**;
- 如未来平台提供 Sora/可灵/即梦等 AI 生成在线工具,需在 **服务上线前 10 个工作日** 通过 `beian.cac.gov.cn` 完成算法备案;
- 备案信息变更需 10 个工作日内更新。

### 3.2 内容审核义务

| 审核对象 | 审核方式 | 审核时长 |
|---|---|---|
| **上架 IP 资产** | AI 明星人脸比对(查重) + 人工审核 | 1-3 工作日 |
| **AIGC 众包终稿** | AI 内容安全(阿里云) + 人工抽检(5%) | 24 小时 |
| **Workspace 中间稿** | 创作者自查 + 平台抽检 | 滚动 |
| **广告主上传素材** | 资质审核 + 内容审核 | 1-3 工作日 |

### 3.3 显著标识义务

依据《生成式人工智能服务管理暂行办法》第 17 条 + 《互联网信息服务深度合成管理规定》第 16、17 条,平台须对 AIGC 内容 **显著标识**:

> **v2 落地动作** — W2 起,平台自动为所有 AIGC 内容(含买家 brief 中由 LLM 拆解的 spec、由创作者在 Workspace 生成的终稿)叠加不可见水印;**短视频与广告片须在视频右下角加 "AI 生成" 角标(至少画面宽度 3%,停留时长 ≥ 3 秒);海报 / KV 须在画面边角加 "AI 生成" 图标**。

### 3.4 通知-删除义务(避风港原则)

依据《电子商务法》第 42-45 条,平台在收到权利人 **有效通知** 后:

1. **24 小时内** 采取下架、删除、屏蔽、断开链接等必要措施;
2. **同步通知** 创作者/买家;
3. 创作者/买家有 **反通知** 权利,反通知合规的,平台恢复内容;
4. 平台对 **明知或应知** 侵权的,承担连带责任。

🚧 平台通知邮箱:`legal-notice@ibi.ren`(模板另行公告)。

### 3.5 留痕义务

平台对以下信息 **保留 ≥ 3 年**:

- 用户实名信息(KYC)
- IP 资产上架记录 + 链上存证哈希
- AIGC 众包订单与 Workspace 消息
- 内容审核记录(谁审、何时审、结论)
- 通知-删除工单

## 4. 创作者义务

### 4.1 原创与权属

详见《用户协议 v2》§3.2 与原 v1.0 《作品原创性及自主承担侵权责任承诺书》第二章。核心要求:

- **本人原创或已取得合法授权**(LoRA 模型、训练数据、声音样本、人物小传);
- **不侵犯明星肖像权、第三方著作权、商标权**;
- 授权链条可追溯,平台有权要求补充证明。

### 4.2 不得制作的 AIGC 内容

🚧 **绝对禁止**(违反即下架 + 信用分扣 30-100 分 + 永久封禁):

1. **政治敏感**:涉及国家领导人、政治人物、政治事件的内容(《网络信息内容生态治理规定》第 6 条);
2. **色情低俗**:淫秽、色情、陪聊、低俗擦边内容;
3. **暴恐血腥**:宣扬恐怖主义、渲染暴力血腥;
4. **明星/真实人物侵权**:未经授权使用可识别自然人(明星、政治人物、网红)形象;
5. **国家机密/军事设施**:任何涉密、涉军内容;
6. **未成年人不良内容**:涉及未成年人的色情、暴力、自残、自杀、厌学、拜金;
7. **虚假信息**:故意编造虚假新闻、虚假科普、虚假证言;
8. **违反公序良俗**:违背社会主义核心价值观的内容。

### 4.3 受限内容(需买家合规自查)

| 行业 | 限制 |
|---|---|
| **医疗** | 需《医疗广告审查证明》;禁止保证治愈率;禁止利用患者形象作推荐 |
| **药品/医疗器械** | 同上,需相应广告批准文号 |
| **金融** | 需相应金融牌照;禁止保本保收益承诺;禁止虚假宣传 |
| **教育培训** | 禁止"包过""必上"承诺;禁止利用未成年人代言 |
| **房地产** | 禁止升值承诺;禁止误导性价格宣传 |
| **酒类** | 禁止诱导饮酒;禁止未成年人形象 |
| **保健食品** | 蓝帽子标志 + "本品不能代替药物" |
| **烟草** | 完全禁止 |

### 4.4 AIGC 显著标识执行细则

| 媒介 | 标识要求 | 平台自动叠加? |
|---|---|---|
| **短视频(横屏 16:9)** | 右下角"AI 生成"角标,宽度 ≥ 画面宽 3%,停留全程 | ✅ |
| **短视频(竖屏 9:16)** | 顶部状态栏下方"AI 生成"标签 | ✅ |
| **广告片 30-60s** | 片头/片尾"本广告由 AI 数字人出演"语音 + 角标 | ✅ |
| **海报 / KV** | 主视觉边角"AI 生成"图标 | ✅ |
| **直播** | 直播间标题前缀"[AI 数字人]" | ✅ |
| **文案 / 标题** | 文末/标题前缀"#AI 生成" | ✅ |

### 4.5 跨平台分发仍负有合规责任

创作者将通过 ibi.ren 制作的 AIGC 内容分发到 **抖音/视频号/小红书/TikTok** 等第三方平台时,仍须遵守各平台的 AIGC 内容规则:

- 抖音:《抖音社区自律公约》AIGC 标识规范
- 视频号:微信平台 AIGC 标注规则
- 小红书:小红书《社区公约》AI 标注规则
- TikTok:TikTok AIGC Disclosure Policy

平台不为第三方平台的违规处置负责,但 **保留对持续违规创作者的信用分扣罚**。

## 5. 买家/广告主义务

### 5.1 广告内容合规自查

买家在平台发布 brief 前,应自行完成以下自查:

- [ ] 涉及医疗/药品/医疗器械/金融/教育/房地产/酒类/保健食品/烟草等 **受限行业**,是否已取得相应资质
- [ ] 涉及的代言人(明星、名人、专家、消费者形象),是否已取得书面授权
- [ ] 涉及的商标/品牌/Logo,是否已取得使用授权
- [ ] 涉及的数据/统计/排名/排名声明,是否有出处
- [ ] 文案中是否含有 **绝对化用语**:"最""第一""国家级""顶级"等(《广告法》第 9 条)
- [ ] 是否涉及 **比较广告**:贬低其他生产经营者的商品或服务(《广告法》第 16 条)
- [ ] 是否含 **诱导性消费**:"扫码""点击链接""立即购买"等(《广告法》第 14 条,需显著标明"广告")

### 5.2 投放前合规审查

| 内容类型 | 是否需法务/合规复核 |
|---|---|
| 一般品牌广告(食品饮料、日用品、家电) | 平台初审即可 |
| **医疗/药品/医疗器械** | **必须**买家法务/合规部出具审核意见 |
| **金融/投资/理财** | **必须**买家法务/合规部出具审核意见 |
| **教育培训(尤其涉及未成年人)** | 建议买家法务/合规部审核 |
| **跨境投放**(港澳台/海外) | **必须**买家跨境合规顾问审核 |
| **代言人含真实人物** | 建议买家审核代言人授权链 |

### 5.3 行业禁词(AI 生成 ≠ 可规避)

> 创作者不应在买家指示下,通过 AIGC 绕开广告法限制。以下行为即便买家指示,平台也将拒绝服务:

- 通过 AIGC 生成 **未授权名人代言**(如"习酒·刘德华代言")
- 通过 AIGC 生成 **虚假证言**(如"用了这款药,我的癌症痊愈了")
- 通过 AIGC 生成 **无资质的医疗/金融承诺**
- 通过 AIGC 生成 **未成年人吸烟饮酒形象**
- 通过 AIGC 生成 **伪造的国家机关/官方机构认证**

## 6. 内容审核红线清单

> 平台审核员快速判定 SOP;详细 SOP 见 `apps/api/src/moderation/` 模块代码。

### 6.1 一票否决(立即下架 + 封禁)

- ❌ 涉及政治敏感、领导人、涉政谣言
- ❌ 明星/真实人物侵权(AI 比对相似度 > 0.85)
- ❌ 色情、暴恐、未成年人不良内容
- ❌ 国家机密、军事设施
- ❌ 烟酒/医疗/金融的虚假承诺
- ❌ 诱导自杀、自残、犯罪

### 6.2 高风险(下架 + 通知整改)

- ⚠️ 行业资质存疑(医疗/金融/教培)
- ⚠️ 代言人授权存疑
- ⚠️ 数据/统计未注明出处
- ⚠️ 涉及比较广告、绝对化用语
- ⚠️ 跨平台分发后被举报

### 6.3 中风险(警告 + 创作者自查)

- ⚠️ AIGC 显著标识缺失
- ⚠️ 文案歧义、易引发误解
- ⚠️ 场景化/剧情化表达擦边

### 6.4 正常通过

- ✅ 一般品牌广告、AIGC 标识规范、行业资质齐全、原创素材
