## 2. 账户注册与使用

### 2.1 注册资格

> **v2 新增** — 较 v1 增加了双角色身份与未成年人禁入条款。

用户须为年满 **18 周岁** 具有完全民事行为能力的自然人,或在中华人民共和国境内合法设立的法人/非法人组织。**未成年人**禁止注册;一经发现,平台有权立即关闭账户并删除数据。

### 2.2 实名认证(KYC)

依据《电子商务法》第二十三条与《网络安全法》第二十四条,所有用户须完成 **实名认证(KYC)** 后方可发布或承接内容:

- **个人用户**:身份证正反面 + 人脸活体验证;
- **企业用户**:营业执照 + 法定代表人身份证 + 对公账户验证;
- **平台当前 KYC 实现**:由于阿里云普通实人认证已退市(详见 memory `project-kyc-alicloud-retired`),平台采用 **人工审核 + 证件 OCR 校验 + 人工活体视频** 三步走 driver,审核时长 1–3 个工作日。

用户承诺提供的 KYC 信息真实、准确、完整,并授权平台对信息进行核实与存储。

### 2.3 双角色与单一账户

- 用户可同时具有 **买家**(buyer)、**创作者**(creator)两种身份,使用同一账户切换。
- 同一自然人不得注册多个账户;一经发现,平台有权合并、限制或关闭相关账户。

### 2.4 账户安全

用户应妥善保管账户密码、API Key、签署设备等身份验证要素;因用户主动泄露、出借账户导致的损失,由用户自行承担;平台在收到账户被盗通知并核实后,将协助冻结账户与争议处理。

### 2.5 AI Agent 身份与责任归属 (#30.7.1 W2 #28,2026-07-01 增)

**2.5.1 双边 Agent 形态**

用户(买家/创作者)可使用本地部署的 AI Agent 工具(包括但不限于 Claude Code、Codex、Cursor、小龙虾龙虾、第三方 SKILL 插件)代表自己完成以下行为:
- 买家 Agent:读取本地需求文件、提炼标准任务包、询价、自动验收
- 创作者 Agent:自动接单、本地生成内容、自检、向对方 Agent 提交

**2.5.2 行为归属原则**

用户本地 Agent 的一切行为,**视为用户本人的行为**,由用户承担全部法律责任。理由:
- Agent 工具由用户选择、部署、配置、运行
- Agent 提示词(Prompt)由用户编写或选用
- Agent 操作的账户、文件、决策均由用户授权
- Agent 不具备独立法律主体资格

**2.5.3 用户义务**

使用本地 Agent 时,用户应:
- 对 Agent 输出做最终人工 review(尤其是涉及合同签署、支付、版权登记、跳单等关键操作)
- 不得用 Agent 从事本协议 §3 禁止的任何行为
- 妥善保管 Agent 工具的访问凭证(API Key 等),等同于账户安全责任

**2.5.4 平台 Agent 责任**

平台提供的 Agent(平台仲裁 Agent / 推送 Agent / AI 拆解 Agent)由平台运营,其行为后果由平台承担,但:
- Agent 判定是"参考",终极争议由 admin /disputes 人工复核
- 平台 Agent 出错导致双边损失的,平台累计赔付上限 = 12 个月服务费(详见 §11.2)

3. 违反《广告法》第九至二十八条的虚假宣传、医疗/金融/未成年人特殊广告限制;

### 3.2 创作者专属规范

1. **原创保证**:上传的所有 IP 资产系本人原创或已取得合法、完整、有效的授权,不侵犯第三方著作权、肖像权、商标权等合法权益(具体条款见原 v1.0 承诺书第二章)。

买家除遵守 §3.1 外,还须:

1. 发布的 brief 真实、合法,不得用于套取创作者方案、压价、跳单。
2. 不得绕过平台与中标创作者直接交易(详见 §7 跳单检测)。
3. 验收应基于事先约定的 spec,不得无理由拒收或拖延付款。

### 4.2 授权模式

IP 资产支持以下授权模式(创作者自行设定):

| 模式 | 含义 | 平台抽成 |
|---|---|---|
| 非独家授权 | 创作者可同时授权他人 | 15% |
| 限时独家授权 | 创作者在授权期内不得授权他人 | 20% |
| 永久买断 | 创作者将本 IP 资产全部权利转让给买家 | 25% |

### 4.3 链上存证

> **v2 新增** — v1 无存证条款。

### 4.4 版权代申请服务

创作者可委托平台代为办理著作权登记(详见 `2026-privacy-policy-v2.md` §6.3 单独告知):

- **方案 A**(默认):著作权登记在创作者本人名下,平台仅作为代办方收取代办费(¥299/件);

## 5. AIGC 众包撮合(L2 业务)

### 5.1 业务模式

> **v2 新增章节** — v1 无此业务。

```
需求方发包(brief) → 平台 LLM 拆解 + 创作者投标 → 买家选中 → 平台创建 Workspace
   → 创作者在 Workspace 完成 AIGC 内容 → 中间稿评论 → 终稿交付 → 买家验收 → 平台担保支付
```

### 5.2 Workspace 履约

- **履约期限**:以 brief 中 `deadlineAt` 为准;超期按 §5.4 处理。
- **中间稿**:创作者在 Workspace 提交中间稿供买家 review,买家应在 **48 小时内** 回复反馈或确认;逾期未回复视为通过。
- **终稿交付**:创作者交付符合 spec 的终稿,买家验收通过后,平台在 **7 个工作日内** 完成结算。

### 5.3 平台抽成与定价

- **平台佣金**:买家支付金额的 **5%** 作为平台服务费(W1 试点价,正式价以 v3 公告为准)。
- **LLM 报价辅助**:买家发包时可使用 `/api/v1/pricing/estimate` 获得三档报价(essential ¥700-1k / standard ¥1.4k-2k / premium ¥2.5-3.5k)作为参考;该报价由 Claude AI 生成,**不构成买卖双方报价的强制依据**。
- **创作者收入**:创作者实际收入 = 买家支付 - 平台佣金 - 第三方支付通道费(微信 0.6% / 支付宝 0.6%)。

### 5.4 违约责任

> **v2 新增** — v1 完全无违约条款。

| 违约情形 | 责任承担 |
|---|---|
| 创作者超期未交付且未申请延期 | 平台代为扣除违约金 = brief 金额的 **10%**,支付给买家;同时扣创作者信用分 10 分 |
| 买家无理由拒收 | 创作者可申请平台介入,经平台判定买家违约的,买家须支付 brief 金额的 **10%** 作为违约金,并扣买家信用分 10 分 |
| 买家拖延付款超 7 日 | 自第 8 日起按日加收 **0.05%** 滞纳金,并可暂停买家发包权限 |
| 创作者交付物明显不符合 spec | 买家可发起争议,平台介入判定;判定为创作者责任的,创作者须返还款项并扣信用分 20 分 |

## 6. 信用分与处罚分级

### 6.1 信用分规则

> **v2 新增** — v1 仅承诺"原创",无量化信用分。

平台对每个账户维护一个 **信用分(Honor Score)**,初始分 100,范围 0–150。具体规则由平台另行公告,核心维度:

- 原创性(侵权举报成立次数)
- 履约率(准时交付率 / 准时付款率)
- 评价(双向 5 星制平均)
- 平台行为(违规扣分)

### 6.2 处罚分级

| 信用分 | 状态 | 限制 |
|---|---|---|
| 120+ | **金牌** | 搜索加权、可承接 premium 档 brief |
| 100-119 | **正常** | 全部功能 |
| 80-99 | **观察** | 每日发包/投标上限 3 次 |
| 60-79 | **黄牌** | 不可承接 premium 档,周接单上限 2 单 |
| <60 | **红牌** | 禁止接单/发包 30 天 |
| <30 或单次重大违规 | **永久封禁** | 不可恢复 |

### 6.3 申诉

用户对信用分扣罚有异议的,可在扣罚通知送达 **15 日内** 提交申诉,平台在 **7 个工作日内** 复核并书面回复。

## 7. 跳单检测与处罚

> **v2 新增** — v1 无此业务自然无跳单条款。W6 #44 落地此机制。

### 7.1 跳单定义

**跳单**指用户通过以下方式绕过平台完成本应通过平台交易的撮合:

1. 在 Workspace / brief 评论中私下交换微信、邮箱、手机号等联系方式;
2. 通过 Workspace 中间稿获取创作者方案后,在平台外单独委托;
3. 引导创作者/买家通过平台外支付(微信/支付宝转账、对公账户直转);
4. 其他绕开平台担保与抽成的撮合行为。

### 7.2 检测机制

平台通过 **关键词扫描**(微信号、手机号、邮箱、对公账户)、**素材水印**(中间稿叠加平台不可见水印,泄露后可追溯)、**履约关联分析** 等技术手段检测跳单行为。

### 7.3 跳单处罚

经平台判定跳单成立的:

- **首次**:警告,扣除信用分 30 分,清空当月推荐位资格;
- **二次**:扣除信用分 80 分,封禁账户 90 天;
- **三次及以上或金额较大**:永久封禁,平台保留追偿已发生佣金损失的权利。
