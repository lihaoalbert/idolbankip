> **敏感个人信息** — 依据 PIPL 第 28 条,处理需取得 **单独同意**。

- **个人 KYC**:姓名、身份证号、身份证正反面照片、人脸活体视频(平台当前采用人工审核 + OCR 校验);

KYC 信息 **仅用于实名审核,不用于任何商业营销或模型训练**。审核完成后,**身份证号、人脸活体视频** 在数据库中以 AES-256-GCM 加密存储(密钥由 KMS 托管),仅审核员可解密查看。

### 3.1 法定处理情形

3. 为应对突发公共卫生事件、保护自然人生命健康或财产安全所必需;
4. 在合理的范围内处理你自行公开或其他已经合法公开的个人信息;

## 4. 我们如何存储与保护你的信息

### 4.1 存储地点

- **中国境内**:阿里云华东 2(上海)ECS + RDS MySQL + OSS 三桶,详见 `docs/deploy.md`;
- **跨境传输**:**默认不跨境**,你的个人信息不出境;
- **CDN 加速**:阿里云 CDN 仅缓存静态资源(图片/视频),不涉及个人信息处理。

### 4.2 存储期限

| 信息类别 | 存储期限 | 到期处理 |
|---|---|---|
| KYC 信息 | 账户存续期间 + 注销后 3 年 | 删除可识别信息,保留加密哈希备查 |
| 订单/支付记录 | **10 年**(依据《会计档案管理办法》) | 法定保留期届满后删除 |
| 操作日志 | 6 个月 | 滚动删除 |
| 链上存证 | 永久(去中心化,平台不可单独删除) | 不可删除 |
| Cookie | 见 §2.5 表格 | 到期自动清除 |

### 4.3 安全措施

- **传输加密**:全站 HTTPS,符合 TLS 1.2+(阿里云 SSL 证书,详见 memory `project-aliyun-ssl-renewal`)
- **存储加密**:KYC 敏感信息 AES-256-GCM 加密,密钥由阿里云 KMS 托管;
- **访问控制**:RBAC 权限分级,审核员仅可访问必要字段;
- **日志审计**:所有敏感数据访问行为写入审计日志(平台 `AuditModule` 已落地);
- **渗透测试**:每年至少 1 次第三方渗透测试;
- **应急响应**:如发生个人信息泄露、损毁或丢失,我们将在 **72 小时内** 通知你与监管机构。

## 6. 你的权利

依据 PIPL 第四章,你享有以下权利:

- 你有权知晓我们如何处理你的个人信息(本政策全文披露);
- 你有权决定是否同意 **敏感个人信息** 处理(KYC、人脸、声音等),拒绝不影响基础功能。

### 6.2 查阅权

- 你可在 **个人中心 → 数据导出** 下载你的账户信息、上传作品、订单、消息记录副本(JSON + ZIP 打包),处理时长 **15 个工作日**;
- KYC 原始证件、人脸视频出于安全原因 **不支持在线查看**,如需调取请联系 `privacy@ibi.ren`。

### 6.3 更正与补充

你可在个人中心直接修改昵称、头像、简介、邮箱、手机号;**KYC 信息一经审核通过不可自行修改**,如需更正请联系客服提交重新审核(15 个工作日)。

### 6.4 删除权

下列情形你可请求删除:

1. 我们的处理目的已实现、无法实现或为实现目的不再必要;
2. 我们停止提供产品或服务,或保存期限已届满;
3. 你撤回同意(限于基于同意的处理);
4. 我们违反法律法规或本政策处理你的个人信息。

申请路径:**个人中心 → 注销账户**(自动触发可识别信息删除)。

### 6.5 撤回同意

- Cookie:浏览器或账户设置中关闭;
- 营销通知:个人中心 → 通知设置 → 关闭营销类;
- 声音样本:撤回后,基于该声音训练的数字人需同步下架。

### 6.6 注销权

- 注销申请路径:个人中心 → 账户安全 → 注销账户;
- 平台在 **15 个工作日内** 完成账户注销与可识别个人信息删除;
- 法定保留信息(订单/支付记录 10 年)按 §4.2 保留;
- 链上存证不可删除(去中心化)。

### 6.7 投诉与诉讼

如你认为我们违反 PIPL 或本政策,可通过以下渠道投诉:

- 平台内部:`privacy@ibi.ren`(15 个工作日内回复);
